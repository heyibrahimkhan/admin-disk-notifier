1. First run devcon.exe with "devcon.exe status GenDisk" without quotes
2. Copy Disk IDs that you want to whiteList and paste them in mUsb.py in the device IDs list
3. Copy mUsb.py - Shortcut
4. Press Win + r
5. Type "shell:startup" without quotes
6. Paste shortcut in the newly opened folder
7. You can now click on the shortcut or restart PC for the effect to occur now
